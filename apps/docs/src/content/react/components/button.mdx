## Import

Import just the button styles.

```tsx
import {
  Button,
  ButtonIcon,
  ButtonLabel,
  ButtonPending,
} from "@winnie-ui/react/button";
```

## Anatomy

Arrange the components in the structure below.

```tsx
<Button>
  <ButtonIcon />
  <ButtonLabel />
  <ButtonPending />
</Button>
```

### Pending

If a child of the button is rendered with the `ButtonPending` slot component, any text and icons rendered in the button will be hidden and the children passed to the `ButtonPending` slot component when the `isPending` prop is `true`.

<Demo>
  <ButtonPendingDemo client:only="react" />
  <Fragment slot="footer">
    ```tsx {3-5} ins=isPending
    <Button isPending>
      ...
      <ButtonPending>
        <Loader />
      </ButtonPending>
    </Button>
    ```
  </Fragment>
</Demo>

## Reference

Refer to React Aria Components [documentation](https://react-spectrum.adobe.com/react-aria/Button.html#props) for more details about the available behavioral props and events available on Button.

| Prop | Type | Default |
| ------------- | ------------------------------------------- | --- |
| `color` | `"brand" \| "red" \| "orange" \| "yellow" \| "green" \| "blue" \| "purple" \| "pink" \| "grey"` | `undefined` |
| `radius` | `"none" \| "sm" \| "md" \| "lg" \| "round"` | `undefined` |
| `size`   | `"sm" \| "md" \| "lg"`                      | `"md"` |
| `width`  | `"auto" \| "full"`                            | `"auto"`|
