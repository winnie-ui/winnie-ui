## Import

Import all of the parts of the button.

```tsx
import {
  Button,
  ButtonIcon,
  ButtonLabel,
  ButtonPending,
} from "@winnie-ui/react/button";
```

## Anatomy

Arrange the components in the structure below.

```tsx
<Button>
  <ButtonIcon />
  <ButtonLabel />
  <ButtonPending />
</Button>
```

If a child of the button is rendered with the `ButtonPending` slot component, any text and icons rendered in the button will be hidden and the children passed to the `ButtonPending` slot component when the `isPending` prop is `true`.

## Reference

Refer to React Aria Components [documentation](https://react-spectrum.adobe.com/react-aria/Button.html#props) for more details about the other props and events available on Button.

| Prop | Type | Default |
| ------------- | ------------------------------------------- | --- |
| `color` | `"brand" \| "red" \| "orange" \| "yellow" \| "green" \| "blue" \| "purple" \| "pink" \| "grey"` | `undefined` |
| `radius` | `"none" \| "sm" \| "md" \| "lg" \| "round"` | `undefined` |
| `size`   | `"sm" \| "md" \| "lg"`                      | `"md"` |
| `variant`   | `"solid" \| "soft" \| "outlined" \| "plain"`                      | `"solid"` |
| `width`  | `"auto" \| "full"`                            | `"auto"`|


| Selector | Description |
| ---------| ------------|
| `data-hovered`  | When the button is hovered with mouse |
| `data-pressed`  | When the button is pressed |
| `data-focused`  | When the button is focused with either a keyboard or mouse |
| `data-focus-visible`  | When the button is focused with keyboard |
| `data-disabled`  | When the button is disabled |
| `data-pending`  | When the button is in a pending state |
