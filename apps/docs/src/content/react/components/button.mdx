## Import

Import just the button styles.

```tsx
import { Button, ButtonIcon, ButtonLabel, ButtonPending } from "@winnie-ui/react/button"
```

## Anatomy

Arrange the components in the structure below.

```tsx
<Button>
	<ButtonIcon />
	<ButtonLabel />
	<ButtonPending />
</Button>
```
