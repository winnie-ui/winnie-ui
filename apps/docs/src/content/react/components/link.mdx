## Import

Import the link component.

```tsx
import { Link } from "@winnie-ui/react/link";
```

## Anatomy

Arrange the components in the structure below.

```tsx
<Link />
```

## Reference

Refer to React Aria Components [documentation](https://react-spectrum.adobe.com/react-aria/Link.html#props) for more details about the other props and events available on Link.

| Prop | Type | Default |
| ------------- | ------------------------------------------- | --- |
| `align` | `"left" \| "center" \| "right"` | `left` |
| `color` | `"brand" \| "red" \| "orange" \| "yellow" \| "green" \| "blue" \| "purple" \| "pink" \| "grey"` | `undefined` |
| `contrast`   | `"low" \| "normal" \| "high"`                      | `"high"` |
| `size`   | `"1" \| "2" \| "3" \| "3" \| "4" \| "5" \| "6" \| "7" \| "8" \| "9"`                      | `"3"` |
| `weight`   | `"normal" \| "medium" \| "semibold" \| "bold" \| "extra-bold"`                      | `"normal"` |

| Selector | Description |
| ---------| ------------|
| `data-hovered`  | When the link is hovered with mouse |
| `data-pressed`  | When the link is pressed |
| `data-focused`  | When the link is focused with either a keyboard or mouse |
| `data-focus-visible`  | When the link is focused with keyboard |
| `data-disabled`  | When the link is disabled |
| `data-current`  | When the link is the current item in a list |
