## Import

Import all of the parts of the link button.

```tsx
import {
  LinkButton,
  LinkButtonIcon,
  LinkButtonLabel,
} from "@winnie-ui/react/link-button";
```

## Anatomy

Arrange the components in the structure below.

```tsx
<LinkButton>
  <LinkButtonIcon />
  <LinkButtonLabel />
  <LinkButtonPending />
</LinkButton>
```

## Reference

Refer to React Aria Components [documentation](https://react-spectrum.adobe.com/react-aria/link.html#props) for more details about the other props and events available on LinkButton.

| Prop      | Type                                                                                            | Default     |
| --------- | ----------------------------------------------------------------------------------------------- | ----------- |
| `color`   | `"brand" \| "red" \| "orange" \| "yellow" \| "green" \| "blue" \| "purple" \| "pink" \| "grey"` | `undefined` |
| `radius`  | `"none" \| "sm" \| "md" \| "lg" \| "round"`                                                     | `undefined` |
| `size`    | `"sm" \| "md" \| "lg"`                                                                          | `"md"`      |
| `variant` | `"solid" \| "soft" \| "outlined" \| "plain"`                                                    | `"solid"`   |
| `width`   | `"auto" \| "full"`                                                                              | `"auto"`    |

| Selector             | Description                                                |
| -------------------- | ---------------------------------------------------------- |
| `data-hovered`       | When the link button is hovered with mouse                      |
| `data-pressed`       | When the link button is pressed                                 |
| `data-focused`       | When the link button is focused with either a keyboard or mouse |
| `data-focus-visible` | When the link button is focused with keyboard                   |
| `data-disabled`      | When the link button is disabled                                |
| `data-current`       | When the link button is the current item in a list
