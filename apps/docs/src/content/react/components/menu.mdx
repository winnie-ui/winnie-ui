## Import

Import all of the parts of the menu. 

```tsx
import {
  MenuProvider,
  MenuPopover,
  Menu,
  MenuItem
} from "@winnie-ui/react/menu";
```

## Anatomy

Arrange the components in the structure below.

```tsx
<MenuProvider>
  <Button />
  <MenuPopover>
    <Menu>
      <MenuItem />
    </Menu>
  </MenuPopover>
</MenuProvider>
```
