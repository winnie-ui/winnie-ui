## Import

Import all of the parts of the menu. 

```tsx
import {
  MenuProvider,
  MenuPopover,
  Menu,
  MenuSeparator,
  MenuItem,
  MenuItemGroup,
  MenuItemLabel,
  MenuItemDescription,
  MenuItemIcon
} from "@winnie-ui/react/menu";
```

## Anatomy

Arrange the components in the structure below.

```tsx
<MenuProvider>
  <Button />
  <MenuPopover>
    <Menu>
      <MenuItem>
        <MenuItemLabel />
        <MenuItemDescription />
      </MenuItem>
      <MenuSeparator />
      <MenuItem>
        <MenuItemGroup>
          <MenuItemLabel />
          <MenuItemDescription />
        </MenuItemGroup>
      </MenuItem>
      <MenuSeparator />
    </Menu>
  </MenuPopover>
</MenuProvider>
```
