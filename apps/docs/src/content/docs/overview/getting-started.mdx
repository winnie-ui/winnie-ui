Alku CSS is a collection of css variables and component styles that can be used to style components in any framework.

## Installation

Let's get you up and running with Alku CSS.

### 1. Install Alku CSS

Using the package manager of your choice, install Alku CSS from your command line:

{/* prettier-ignore */}
<Tabs>
  <TabItem label="npm">
    ```bash frame="none"
    npm install @alku/css
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash frame="none"
    yarn add @alku/css
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash frame="none"
    pnpm add @alku/css
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash frame="none"
    bun install @alku/css
    ```
  </TabItem>
</Tabs>

### 2. Import the CSS

Import the Alku CSS file at the root of your application in the place of your choice:

{/* prettier-ignore */}
<Tabs>
  <TabItem label="React">
    ```tsx title="main.tsx"
    import "@alku/css";
    ```
  </TabItem>
  <TabItem label="CSS">
    ```css title="global.css"
    @import "@alku/css";
    ```
  </TabItem>
</Tabs>

### 3. Configure Alku CSS

Configure the `body` element with Alku CSS specific attributes

```tsx title="root.tsx" ins="class="alku"" ins="data-accent-color="blue"" ins="data-scaling="100%""
export default function Root() {
  return (
    <html>
      <head></head>
      <body class="alku" data-accent-color="blue" data-scaling="100%">
        <App />
      </body>
    </html>
  );
}
```

### 4. Start Building

Your app is now ready to go with Alku CSS.

```css title="my-button.css"
.my-button {
  background: var(--ak-accent-color-9);
  color: var(--ak-accent-color-contrast);
  height: 32px;
  padding-inline: var(--ak-space-3);
}
```
