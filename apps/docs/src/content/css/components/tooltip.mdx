## Import

Import just the tooltip styles.

```css
@import "@winnie-ui/component/tooltip";
```

## Anatomy

Attach the appropriate `data` attributes in the structure below.

```html
<div data-component="tooltip">
  <span data-slot="label"></span>
  <kbd data-slot="shortcut"></kbd>
</div>
```

## Examples

### Radius

Use the `data-radius` attribute to change the radius of the tooltip.

<Demo>
	<div data-component="tooltip" data-radius="none" class="tooltip-demo" style={{ marginRight: "var(--wui-space-3)" }}>
    <span data-slot="label">View Dish</span>
	</div>
	<div data-component="tooltip" data-radius="md" class="tooltip-demo" style={{ marginRight: "var(--wui-space-3)" }}>
    <span data-slot="label">View Dish</span>
	</div>
	<div data-component="tooltip" data-radius="round" class="tooltip-demo">
    <span data-slot="label">View Dish</span>
	</div>
	<Fragment slot="footer">
    ```html ins=data-radius="none" ins=data-radius="md" ins=data-radius="round"
    <div data-component="tooltip" data-radius="none">
    	<span data-slot="label">View Dish</span>
    </div>
    <div data-component="tooltip" data-radius="md">
    	<span data-slot="label">View Dish</span>
    </div>
    <div data-component="tooltip" data-radius="round">
    	<span data-slot="label">View Dish</span>
    </div>
    ```
	</Fragment>
</Demo>

### Shortcuts

Use the `shortcut` slot as a child of the tooltip to show a keyboard shortcut.

<Demo>
	<div data-component="tooltip" class="tooltip-demo">
    <span data-slot="label">Delete Dish</span>
    <kbd data-slot="shortcut">
      <kbd>⇧</kbd>
      <kbd>⌘</kbd>
      <kbd>⌫</kbd>
    </kbd>
	</div>
	<Fragment slot="footer">
    ```html ins=data-slot="shortcut"
    <div data-component="tooltip">
    <span data-slot="label">Delete Dish</span>
	    <kbd data-slot="shortcut">
		    <kbd>⇧</kbd>
		    <kbd>⌘</kbd>
		    <kbd>⌫</kbd>
	    </kbd>
    </div>
    ```
	</Fragment>
</Demo>

### Multiline

While long tooltips are supported by Winnie, we do not recommend them as a UI practice. Ideally, tooltips are short, descriptive text used to describe the intent of an action or subject.

<Demo>
	<div data-component="tooltip" class="tooltip-demo">
    <span data-slot="label">
	    Today's Special:
	    Grilled Lemon Herb Chicken with seasonal vegetables.
	    Served with a side of garlic mashed potatoes.
	    Click to view the full menu!
    </span>
	</div>
	<Fragment slot="footer">
    ```html

	    <span data-slot="label">
		    Today's Special:
		    Grilled Lemon Herb Chicken with seasonal vegetables.
		    Served with a side of garlic mashed potatoes.
		    Click to view the full menu!
	    </span>
    </div>
    ```

</Demo>

## Reference

Below is a list of the available tooltip `data-` attributes that are available.

| Attribute     | Value                                       |
| ------------- | ------------------------------------------- |
| `data-radius` | `"none" \| "sm" \| "md" \| "lg" \| "round"` |
