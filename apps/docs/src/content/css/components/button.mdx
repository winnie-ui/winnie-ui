## Import

Import just the button styles.

```css
@import "@winnie-ui/component/button";
```

## Anatomy

Attach the appropriate `data` attributes in the structure below.

```html
<button data-component="button">
  <i data-slot="icon"></i>
  <span data-slot="label"></span>
  <span data-slot="loading"></span>
</button>
```

## Examples

### Size

Use the `data-size` attribute to change the size.

<Demo>
  <ButtonSize />
  <Fragment slot="footer">
    ```tsx ins=data-size="sm" ins=data-size="md" ins=data-size="lg"
    <button data-component="button" data-size="sm">
      <span data-slot="label">New Issue</span>
    </button>
    <button data-component="button" data-size="md">
      <span data-slot="slot">New Issue</span>
    </button>
    <button data-component="button" data-size="lg">
      <span data-slot="slot">New Issue</span>
    </button>
    ```
  </Fragment>
</Demo>

### Radius

Use the `data-radius` attribute to change the radius.

<Demo>

### With Icon

Add an icon with the `icon` slot attribute as a child of the button.

<Demo>
  <ButtonHero />
  <Fragment slot="footer">
    ```tsx
    <button data-component="button">
      <i class="organization" data-slot="icon"></i>
      <span data-component="text">Create Organization</span>
    <button>
    ```
  </Fragment>
</Demo>

### Icon Only

Create an icon button by only passing an icon with the `icon` slot attribute as a child of the button.

<Demo>
  <ButtonIcon />
  <Fragment slot="footer">
    ```tsx
    <button data-component="button">
      <i class="plus" data-slot="icon"></i>
    <button>
    ```
  </Fragment>
</Demo>

### Loading

If a child of the button is rendered with the `loading` attribute, any text and icons rendered in the button will be hidden and the `loading` slot will be rendered.

<Demo>
  <ButtonLoading />
  <Fragment slot="footer">
    ```tsx ins=data-slot="loading"
    <button data-component="button">
      <span data-slot="loading">
        <i class="loader"></i>
      </span>
      <span data-slot="label">New Issue</span>
    </button>
    ```
  </Fragment>
</Demo>

### Full Width

Use the `data-width` attribute to modify the overall width of the button.

<Demo>
  <ButtonWidth />
  <Fragment slot="footer">
    ```tsx ins=data-width="full"
    <button data-component="button" data-width="full">
      <span data-slot="slot">Log in</span>
      <i class="arrow-right" data-slot="icon"></i>
    </button>
    ```
  </Fragment>
</Demo>

## Reference

Below is a list of the available `data-` attributes available for Button.

| Attribute     | Value                                       |
| ------------- | ------------------------------------------- |
| `data-radius` | `"none" \| "sm" \| "md" \| "lg" \| "round"` |
| `data-size`   | `"sm" \| "md" \| "lg"`                      |
| `data-width`  | `"full"`                                    |
