## Import

Import only the field styles.

```css
@import "@winnie-ui/component/field";
```

## Anatomy

Attach the appropriate component and slot attributes in the structure below.

```html
<div data-component="field">
  <label data-slot="label" />
  <span data-slot="description" />
  <!-- control component -->
  <span data-slot="error" />
</div>
```
