## Import

Import just the checkbox styles.

```css
@import "@winnie-ui/component/checkbox";
```

## Anatomy

Attach the appropriate `data-component` attributes in the structure below.

```html
<button data-component="checkbox">
  <i data-component="indicator"></i>
</button>
```

## Examples

### Size

Use the `data-size` attribute to change the size.

<Demo>

## Reference

Below is a list of the available `data-` attributes available for Checkbox.

| Attribute     | Value                                       |
| ------------- | ------------------------------------------- |
| `data-radius` | `"none" \| "sm" \| "md" \| "lg"`            |
| `data-size`   | `"sm" \| "md" \| "lg"`                      |
