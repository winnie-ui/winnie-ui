---
title: Keyframes
description: Variety of animation keyframes for elements
---

## Import

Import just the animation variables.

```css
@import "@winnie-ui/base/animation";
```

## Reference

Below is a list of the available keyframes in Winnie CSS.

| Variable                | Tailwind Class              |
| ----------------------- | --------------------------- |
| `wui-collapsible-down`  | `animate-collapsible-down`  |
| `wui-collapsible-up`    | `animate-collapsible-up`    |
| `wui-fade-in`           | `animate-fade-in`           |
| `wui-fade-out`          | `animate-fade-out`          |
| `wui-slide-up-enter`    | `animate-slide-up-enter`    |
| `wui-slide-up-exit`     | `animate-slide-up-exit`     |
| `wui-slide-down-enter`  | `animate-slide-down-enter`  |
| `wui-slide-down-exit`   | `animate-slide-down-exit`   |
| `wui-slide-left-enter`  | `animate-slide-left-enter`  |
| `wui-slide-left-exit`   | `animate-slide-left-exit`   |
| `wui-slide-right-enter` | `animate-slide-right-enter` |
| `wui-slide-right-exit`  | `animate-slide-right-exit`  |
