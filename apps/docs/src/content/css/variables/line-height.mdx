---
title: Line Height
description: Variables for modifying the line height of an element
---

## Import

Import just the typography variables.

```css
@import "@winnie-ui/base/typography";
```

## Reference

Below is a list of the available line height in Winnie CSS.

| Variable              | Tailwind Class |
| --------------------- | -------------- |
| `--wui-line-height-1` | `leading-1`    |
| `--wui-line-height-2` | `leading-2`    |
| `--wui-line-height-3` | `leading-3`    |
| `--wui-line-height-4` | `leading-4`    |
| `--wui-line-height-5` | `leading-5`    |
| `--wui-line-height-6` | `leading-6`    |
| `--wui-line-height-7` | `leading-7`    |
| `--wui-line-height-8` | `leading-8`    |
| `--wui-line-height-9` | `leading-9`    |
